工具¶

本章中的函数执行各种实用工具任务，包括帮助 C 代码提升跨平台可移植性，在 C 中使用 Python 模块，以及解析函数参数并根据 C 中的值构建 Python 中的值等等。

  • 操作系统实用程序
  • 系统功能
  • 过程控制
  • 导入模块
  • Data marshalling support
  • 语句解释及变量编译
    • 解析参数

      • 数字
      • 其他对象
      • API 函数
    • 创建变量
  • 字符串转换与格式化
  • 反射
  • 编解码器注册与支持功能
    • Codec lookup API
    • Registry API for Unicode encoding error handlers
