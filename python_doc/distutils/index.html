作者

Greg Ward ， Anthony Baxter

电子邮箱

distutils-sig@python.org

参见

分发 Python 模块

最新的模块分发文档

本文档从模块开发人员的角度描述了 Python Distribution Utilities（“Distutils” ，描述了如何使用 Distutils 轻松地为更广泛的受众提供 Python 模块和扩展，而构建/发布/安装机制的开销很小。

注解

本指南仅介绍构建和分发扩展的基本工具，这些扩展是作为此Python版本的一部分提供的。 第三方工具提供更易于使用和更安全的替代方案。有关详细信息，请参阅 Python 打包用户指南中的 快速推荐部分 。
